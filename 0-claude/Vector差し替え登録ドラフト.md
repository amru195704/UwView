## 3. ソフト詳細説明（詳細紹介文）

```
UwView は、2億行クラス（設計上の想定。実際の行数限界は不明）の巨大テキスト
ファイルを省メモリ・高速に閲覧できる、閲覧専用の軽量ビューアです。
一般的なテキストエディタでは開けない大容量ログや、RDB／XML の
ダンプ、点群(XYZ)・GML／CityGML、GNSS(RINEX) など膨大な行数のデータを、
待たされずに開いて見ることに特化しています。

メモリマップドファイルとスパース行インデックスにより、ファイル全体をメモリへ
読み込むことなく、開いた瞬間から表示できます（ページモード）。背景で行インデ
ックスを構築し、完了後は行番号ジャンプが可能な行モードへ自動で切り替わるため、
巨大ファイルでも「待たされない」操作感を実現しました。

主な特徴:
・2億行クラスの巨大テキストを省メモリで高速表示（実測8.9億行・約51GBまで確認）
・開いた瞬間に表示（ページモード）→ 背景索引 → 行モードへ自動昇格
・文字コード自動判定（UTF-8 / Shift-JIS / EUC-JP）＋手動切替
・複数ファイルをタブで切り替えて閲覧
・測量・地図・3D点群など大容量データの確認に好適

本ソフトは、かつて Windows 向けに公開していた UwView（最大5000万行）を
Avalonia で再構築し、Windows／macOS／Linux のクロスプラットフォーム対応・
2億行クラスへ拡張したものです。実測では OpenStreetMap 日本全体を XML 化した
約51GB・8.9億行のファイルまで閲覧を確認しており、実際の行数限界は未知です。
macOS／Linux 版およびソースコードは GitHub で公開しています。

GitHub: https://github.com/amru195704/UwView
```
