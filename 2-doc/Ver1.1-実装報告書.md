## 2. 色分けハイライタ（本丸）

検索と独立に、登録した複数パターンを常時多色着色。klogg の Highlighter 相当 ＋ UwView 資産での上積み。

### コアエンジン ＆ 永続化（UwView.Core / AppSettings）

- **マッチャ**（`CompiledHighlighter`）— 規則を下→上に適用し先頭規則が上書き勝ち。行全体／マッチ部、大小無視、正規表現のキャプチャ部のみ着色に対応
- **検索マッチ最優先**の描画順（色分けの背景の上に検索の黄、マッチ部の文字色は既定へ戻し必ず視認可）。可視行のみ再評価で巨大ファイルでも影響なし
- **32色パレット**（色覚配慮・相互に離した配列）
- **永続化** `AppSettings.Highlighters` ／ **エクスポート・インポート**（`.uwvhl`・同一IDスキップ）
- **単体テスト10件**追加（Core 合計 50 件全緑）

### 管理ダイアログ（HighlighterWindow）

- 規則の**追加・削除・並べ替え**（並べ替え▲▼は行の先頭、削除は ✕）。追加のたびに背景色を**自動で色循環**（先頭パレットを N=10 でループ）、文字色は既定 `#000000`
- **色プレビュー box**（「AaBb」を文字色×背景色で表示）＋各色欄に**明るく／暗く ▲▼**（各RGB値 ±`#10`）
- **名前付きセットの保存・読込** — 「保存」で名前を尋ね（候補に「デフォルト」）、スナップショットとして保持。「読込」で編集を破棄して保存内容へ復元。作業コピー方式で保存内容が編集で壊れない
- **同梱プリセット5種**: 汎用ログレベル／syslog・Linux／Web access (HTTP)／JSON ログ／GNSS NMEA。選ぶと規則を追加、選択エリアに名称を表示。「なし（全てクリア）」も選択可

対応: **UVF / UVP 両対応**・**klogg §2 パリティ**

## 3. 検索・選択系の改善

| 機能 | 内容 | 対応 |
|------|------|------|
| 検索履歴・定義済みフィルタ | 検索ボックスを履歴オートコンプリート化（上限50）。「定義済み」ドロップダウンで即検索、「★」で現在の検索を名前付き保存 | UVF / UVP |
| 検索の次へ／前へ | 該当行を**画面中央**に寄せ、**行全体を強調**（水色）。直近ヒット位置を基準に前後へ確実に移動 | UVF / UVP |
| 行ジャンプ（Go） | 行番号ジャンプも検索移動と同じく中央寄せ＋行強調 | UVF / UVP |
| 結果一覧ポップアップ | 検索完了でヒットありなら自動表示。「閉じる」ボタン、前後±N（UVF=±1／UVP=±1000）、選択行の copy／save（元行番号付き） | UVF / UVP |
| 行選択 / 矩形選択 | 本文ドラッグの行選択は両対応。**矩形選択は UVP 限定**（メイン画面・結果ポップアップのトグル）。右クリックで copy／save | 行のみ UVF ／ 矩形 UVP |
| アプリ名・About・配色 | アプリ名を UwView / UwView Pro に。About に GitHub リンク（`github.com/amru195704`）。配色は UVF=白／UVP=淡いミント緑 | UVF / UVP |

## 5. 未実装・次のステップ

指示書の残り。永続化フィールド（履歴・per-file 状態・セッション・recent・favorites）は
`AppSettings` に準備済みで、着手時の土台はできている。

- **クイックカラーラベル付与UI（Ctrl+Shift+1–9 ほか）** — 選択文字列への即着色。本文のサブ行テキスト選択が前提のため、TextView への機能追加が必要
- **Pro +α ①–⑥** — ミニマップ多色分布・規則ごとの件数バッジ・結果ポップアップ統合・色単位の次へ/前へ・`.uwvz` による全体集計の高速化（同梱プリセットは先行実装済み）
- **その他4機能 B / C / D** — B: Follow 中の増分検索自動更新／C: ファイルごとの状態復元／D: セッション復元・最近使ったファイル・お気に入り
